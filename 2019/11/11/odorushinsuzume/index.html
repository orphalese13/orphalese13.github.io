SP：新城管理官与那之后的故事

发表于 2019-11-11 更新于 2024-09-11 分类于 踊る大捜査線 Disqus：
本文字数： 1.1k 阅读时长 ≈ 1 分钟
  • 脑了个新堇都在给认识的人挑新年礼物时遇到的梗（两人未交往）
  • 可以算作是另一篇的后续



“为什么你要给室井先生买礼物？”新城质问——几乎是逼问道，用的是堇也十分熟悉的审讯犯人的语气。他面色不善。而堇心中则在大叫“因为我很可能会在拜访青岛家时见到他！”

最终新城挑选了看起来就很昂贵的钢笔，而堇捏紧了她买的微不足道的小物件（当然，好好地用礼盒包装了起来），还有商场的美食导览手册。“你，呃，”她出声，意识到自己正无自觉地跟着对方走，令人惊奇的是新城居然始终没有不耐烦地把她挥开，或者皱眉厉斥“不要跟着我”。

“我应该也送你点什么吗？”她问，考虑着既然他们正好在这个关口上遇到。

新城猛地转过身，“这就不用你操心了。”他就是有办法把社交辞令也说得很尖刻，“祝新年快乐。”

“不行，那我至少也得请你吃个东西。”堇冲口而出，“地下一层有家新开的秋田烤米棒专卖店，可以尝试到很多不同的新口味，比如蛋黄酱或者薄荷巧克力，就像香蕉棒那样——”

“秋田烤米棒？”新城紧紧盯着她的样子像蛇盯着青蛙，让她感到了困窘和尴尬。

“嗯，你不是喜欢室井管理官吗？可以趁机贴近一下他的生活喜好嘛。”堇赶紧说，但立刻就后悔了，因为新城的表情就像飓风过境一样七零八落。他盯着她的眼神转为了震惊。

“哎呀，我可以理解啦，”堇试着补救，“他毕竟是个omega，虽然信息素对beta影响没那么大，但他确实还挺好闻的——”

“挺好闻的。”新城重复道，用一种阴暗的神色撇了下嘴角，忽然转身就走。

“喂——等一下啊！”堇追上他——这很容易，因为他们一齐被挡在了商场铜墙铁壁般的人潮里。“好啦，那你到底喜欢什么？”

新城的嘴角抽了抽，“你真的想知道？”

“当然。”不想。

“那就什么也别买，明天晚上跟我出去。”

这句话让恩田堇愣了好几秒来反应。等她终于回过神了，她感觉自己脸红了，“真的？”

听见自己的语调和嗓音，她脸红更厉害了。新城面无表情，“我像是会拿这种事开玩笑吗？”

“其实吧——”

“想好了再回答。”

堇闭上了嘴。她发现自己的美食导览手册不知怎么到了新城手里。对方正展开它，皱着眉头假装仔细研究。他熟练地隐藏在波澜不惊若无其事之后的焦虑让答复他忽然变得异常容易，“好吧，你会请我吃什么？”

新城似乎松了口气，尽管他并没有表现出来。只是说：“我七点钟去接你，穿得正式点。”

堇开始飞快地盘算她要不要借口溜走然后去买点新的化妆品。或者礼服。天啊，或者一件小礼服。但新城把导览地图折了回去，塞回她手里。在他们肌肤相触的那个瞬间，他滚烫的指尖就是她所能感知到的全部。

“现在，你说的那家店在哪里？”他问。




Fin.
